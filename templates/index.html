Konfiguracja LLM i API

Skonfiguruj silnik analizy LLM oraz połączenia API lub lokalne.

Embeddingi pozostają wykonywane przez Gemini. Ten wybór dotyczy tylko walidacji zapożyczeń i konsolidacji uzasadnień.

Klucz API jest bezpiecznie przechowywany w lokalnej bazie SQLite. Priorytet ma zmienna środowiskowa `GEMINI_API_KEY`.

0.10

Dotyczy tylko providera Gemini. Tokeny myślowe są rozliczane jak tokeny wyjściowe. Dla masowej walidacji segmentów zwykle wystarcza minimal lub low.
